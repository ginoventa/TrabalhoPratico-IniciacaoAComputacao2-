# Os arquivos que nessa pasta estão, são fruto de um projeto final individual da 
disciplinas de Iniciação a Computação II ministrada pelo Docente Renato Tinós 
em 2024 no curso de Ciência da Computação da FFCLRP - USP. 

# Eles tem como objetivo final, implementar diferentes algoritmos de ordenação. 
Apesar de não demonstrado aqui, o projeto tinha como um de seus pontos-chave a 
análise das diferentes performances de cada um deles, onde fora definido o mais 
e menos eficiente. 

DISCLAIMER: Os arquivos abrigam algoritmos que foram integralmente implementados e comentados por Giovanna Noventa.
